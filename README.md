# Krishna Vihari Arts

Creative art works and professional art services by Krishna Vihari.

## Project Overview
This is a static portfolio and services website for Krishna Vihari Arts.

## Technologies
- React
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
